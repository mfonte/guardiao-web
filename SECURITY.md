# Política de Segurança — guardiao-web

## Reportar vulnerabilidade

**NÃO abra issue pública.**

Use [Security Advisories](https://github.com/mfonte/guardiao-web/security/advisories/new) ou email `manoel@guardiaoiot.com.br`.

## Escopo

Em escopo:

- XSS / injeção via Realtime DB
- Bypass de roles/permissões no client
- API keys vazadas no bundle
- Auth desconfigurado (allowing anonymous, etc)

Fora de escopo:

- Bugs em dependências já reportados upstream
- Issues que exigem credenciais válidas + acesso físico ao dispositivo do usuário

## Tempo de resposta

7 dias para acknowledgement, 48h para vulnerabilidades críticas.
